Important exported types include:

- `GoogleMapsEnvelope`
- `GoogleMapsIntent`
- `GoogleMapsMode`
- `LocationData`
- `PlaceData`
- `RouteData`
- `QueryData`
- `MapViewData`
- `ResolvedGoogleMapsUrl`
- `RawArtifacts`
- `CoordinateSource`
- `ResolutionStatus`

The library favors discriminated unions and narrow string literals over loose bags of data.
